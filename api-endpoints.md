# Instagram Scraper Stable API - Endpoints

## Host
`instagram-scraper-stable-api.p.rapidapi.com`

## User Endpoints
- `POST /get_ig_user_posts.php` - User Posts
- `POST /get_ig_user_reels.php` - User Reels  
- `POST /ig_get_fb_profile_v3.php` - Account Data V2 (funktioniert!)

## Parameter
- `username_or_url` - Instagram username oder URL

## Headers
- `Content-Type: application/x-www-form-urlencoded`
- `x-rapidapi-host: instagram-scraper-stable-api.p.rapidapi.com`
- `x-rapidapi-key: YOUR_API_KEY`
